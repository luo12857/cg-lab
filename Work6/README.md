# 计算机图形学实验六：可微渲染与 3D 模型优化

**罗晓政202411081010**

## 实验目的

本实验通过可微渲染技术，实现将一个简单的球体优化变形为复杂的 3D 模型（如奶牛模型），主要目的包括：

1. 理解可微渲染（Differentiable Rendering）的基本原理
2. 掌握基于梯度的 3D 模型优化方法
3. 学习使用 PyTorch3D 库进行 3D 渲染和优化
4. 理解多视角剪影损失（Silhouette Loss）在 3D 重建中的应用

## 实现原理

### 可微渲染管道

1. **3D 模型表示**：使用网格（Mesh）表示 3D 模型，由顶点（Vertices）和面（Faces）组成
2. **可微栅格化**：将 3D 模型投影到 2D 图像平面，进行可微栅格化
3. **损失计算**：比较预测剪影与目标剪影的差异
4. **反向传播**：通过梯度下降优化顶点位置

### 损失函数

总损失由以下部分组成：

```
Total Loss = Silhouette Loss + λ₁ × Laplacian Smoothing + λ₂ × Edge Loss + λ₃ × Normal Consistency
```

| 损失项 | 权重 | 作用 |
|--------|------|------|
| Silhouette Loss | 1.0 | 最小化剪影差异 |
| Laplacian Smoothing | 1.0 | 保持网格平滑 |
| Edge Loss | 0.1 | 保持边缘长度均匀 |
| Normal Consistency | 0.01 | 保持法向量一致 |

### 摄像机配置

- 20 个视角围绕模型均匀分布（-180° 到 180°）
- 每个视角渲染一张剪影图
- 使用正交投影，固定距离 2.7

## 实现的功能

- ✅ 从 .obj 文件加载 3D 模型
- ✅ 多视角剪影渲染
- ✅ 可微栅格化
- ✅ 梯度下降优化
- ✅ 网格正则化（拉普拉斯平滑、边缘损失、法向量一致性）
- ✅ 中间结果保存为 .obj 文件


## 使用方法

### 1. 准备模型文件

确保 `cow.obj` 文件与代码文件在同一目录下。如果文件在下载目录，可以复制到当前目录：

```bash
copy "%USERPROFILE%\Downloads\cow.obj" .
```

### 2. 运行代码

```bash
python 1.py
```

### 3. 查看输出

程序运行后会在当前目录创建 `output_meshes` 文件夹，保存 300 次迭代中每 20 步的中间结果：

```
output_meshes/
├── mesh_epoch_000.obj
├── mesh_epoch_020.obj
├── mesh_epoch_040.obj
├── ...
└── mesh_epoch_299.obj
```

### 4. 使用 MeshLab 查看结果

可以使用 [MeshLab](https://www.meshlab.net/) 打开生成的 .obj 文件，查看模型优化过程。

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `num_views` | 20 | 渲染视角数量 |
| `image_size` | 256 | 输出图像分辨率 |
| `epochs` | 300 | 优化迭代次数 |
| `lr` | 1.0 | 学习率 |
| `momentum` | 0.9 | SGD 动量 |

## 技术亮点

1. **端到端可微**：整个渲染和优化流程都是可微的，支持梯度反向传播
2. **多视角约束**：同时使用 20 个视角的剪影，提供更丰富的监督信号
