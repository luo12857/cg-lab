# 光线追踪 (Ray Tracing) — Whitted Style

**罗晓政202411081010**

基于 **Taichi** 编程框架实现的 GPU 加速 Whitted 风格光线追踪器，包含实时交互 GUI。

## 效果展示

![光线追踪演示](光线追踪.gif)

> GIF 展示：光源位置可调，玻璃折射球与镜面反射球的实时渲染效果，边缘平滑（MSAA 抗锯齿）。

## 场景构成

| 物体 | 类型 | 位置 | 材质 |
|------|------|------|------|
| 玻璃球 | 球体 | `(-1.2, 0.0, 0.0)` | 玻璃 (Glass, 折射率 1.5) |
| 银色球 | 球体 | `(1.2, 0.0, 0.0)` | 镜面反射 (Mirror) |
| 地板 | 无限大平面 | `y = -1.0` | 漫反射 + 棋盘格纹理 |

- 摄像机位置: `(0.0, 1.0, 5.0)`，略向下俯视
- 光源: 可移动的点光源（交互式控制）

## 运行方法

### 环境要求

- Python 3.8+
- [Taichi](https://github.com/taichi-dev/taichi) — GPU 编程框架

### 安装依赖

```bash
pip install taichi
```

### 启动

```bash
python 光线追踪.py
```

## 交互控制

程序启动后会打开一个实时渲染窗口，右侧提供 **Controls** 面板：

| 控件 | 说明 | 范围 |
|------|------|------|
| **Light X** | 光源 X 轴位置 | `-5.0 ~ 5.0` |
| **Light Y** | 光源 Y 轴位置（高度） | `1.0 ~ 8.0` |
| **Light Z** | 光源 Z 轴位置 | `-5.0 ~ 5.0` |
| **Max Bounces** | 光线最大弹射次数 | `1 ~ 5` |

调节光源位置可以实时观察阴影和光照变化；增加弹射次数可让镜面球的反射更丰富。

## 程序架构

```
光线追踪.py
├── 场景定义
│   ├── intersect_sphere() — 球体求交
│   ├── intersect_plane()  — 无限大平面求交
│   └── scene_intersect()  — 场景遍历，寻找最近交点
├── 光学函数
│   ├── reflect()    — 镜面反射方向计算
│   ├── refract()    — 斯涅尔定律折射方向计算
│   └── fresnel()    — 菲涅尔反射率近似 (Schlick)
├── 光照计算
│   ├── 镜面材质 — 反射光线递归追踪
│   ├── 玻璃材质 — 折射/反射混合 + 全反射处理
│   ├── 漫反射材质 — Phong 光照模型 + 硬阴影检测
│   └── 阴影射线 — 从交点向光源发射 Shadow Ray
├── 渲染管线
│   ├── MSAA 抗锯齿 — 每像素多次随机采样 + 平均
│   ├── render()  — Taichi Kernel，GPU 并行执行
│   └── 色调映射 — clamp 到 [0, 1]
└── 交互 GUI (ti.ui.Window)
    ├── 光源位置滑块
    └── 最大弹射次数滑块
```

## 技术要点

- **GPU 并行**: 所有像素的光线追踪由 Taichi Kernel 并行计算，充分利用 GPU 算力
- **Whitted 风格递归**: 镜面反射材质支持多次光线弹射（通过迭代模拟递归）
- **斯涅尔定律折射**: 玻璃材质根据折射率计算透射光线方向，支持全反射
- **菲涅尔效应**: 使用 Schlick 近似计算反射/折射概率，真实模拟玻璃表面
- **MSAA 抗锯齿**: 每像素多次随机采样，平滑物体边缘过渡
- **法线偏移**: 每次生成新光线时添加 `1e-4` 偏移量，防止自相交
- **棋盘格纹理**: 基于世界坐标的奇偶性程序化生成
- **硬阴影**: 通过 Shadow Ray 检测光源遮挡
