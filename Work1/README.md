## 项目简介

这是一个基于 Taichi 框架的粒子系统实验项目，实现了一个交互式的鼠标引力效果。当用户在窗口中移动鼠标时，粒子会受到鼠标位置的吸引，形成动态的粒子群效果。

## 项目架构

项目采用模块化设计，主要包含以下文件：

```
CG-Lab/
├── src/
│   └── Work0/
│       ├── config.py      # 配置参数
│       ├── physics.py     # 物理计算
│       └── main.py        # 主程序入口
├── README.md             # 项目说明
└── pyproject.toml        # 项目依赖
```

## 代码逻辑

### 1. 配置模块 (config.py)

定义了系统的各项参数，包括：
- 粒子数量
- 物理参数（引力强度、阻力系数、反弹系数）
- 渲染参数（窗口分辨率、粒子大小、颜色）

### 2. 物理模块 (physics.py)

实现了粒子系统的核心物理逻辑：
- 使用 Taichi 定义显存中的粒子位置和速度场
- 初始化粒子的随机位置
- 实现粒子的物理更新：
  - 计算鼠标对粒子的引力
  - 应用空气阻力
  - 处理边界碰撞

### 3. 主程序模块 (main.py)

负责程序的启动和渲染：
- 初始化 Taichi 环境
- 编译 GPU 内核
- 创建 GUI 窗口
- 进入渲染主循环：
  - 获取鼠标位置
  - 调用物理更新函数
  - 绘制粒子
  - 显示窗口

## 实现功能

1. **鼠标引力效果**：粒子会受到鼠标位置的吸引，跟随鼠标移动
2. **边界碰撞**：粒子碰到窗口边界时会反弹
3. **GPU 加速**：使用 Taichi 框架的 GPU 并行计算，实现高效的粒子模拟
4. **实时交互**：用户可以通过移动鼠标实时影响粒子行为

## 运行效果

当运行程序时，会出现一个窗口，窗口中的粒子会受到鼠标的吸引，形成类似引力场的效果。粒子数量可以通过修改 `config.py` 中的 `NUM_PARTICLES` 参数来调整，以适应不同性能的设备。

## 技术栈

- **Python**：主要编程语言
- **Taichi**：用于 GPU 加速的并行计算框架
- **GUI**：Taichi 内置的图形界面库

## 运行方法

1. 安装 Python 环境
2. 安装项目依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行程序：
   ```bash
   python src/Work0/main.py
   ```
