# 计算机图形学实验七：质点弹簧模型

## 实验目的

本实验实现了基于质点弹簧模型的布料模拟系统，主要目的包括：

1. 理解质点弹簧模型的基本原理
2. 实现三种数值积分方法（显式欧拉、半隐式欧拉、隐式欧拉）并对比其稳定性
3. 完善弹簧模型，添加剪切弹簧和弯曲弹簧
4. 实现布料与球体的碰撞检测与响应

## 实现原理

### 质点弹簧模型

布料由 N×N 的质点网格构成，质点之间通过弹簧连接。每个质点受到以下力的作用：

- **重力**：`F_gravity = m * g`
- **阻尼力**：`F_damping = -k_d * v`（与速度成正比，方向相反）
- **弹簧力**：`F_spring = -k_s * (dist - rest_length) * normal`（胡克定律）

### 数值积分方法

1. **显式欧拉（Explicit Euler）**：极易发散，仅作对比演示
   ```
   x_new = x_old + v_old * dt
   v_new = v_old + (F/m) * dt
   ```

2. **半隐式欧拉（Semi-Implicit Euler）**：相对稳定，默认使用
   ```
   v_new = v_old + (F/m) * dt
   x_new = x_old + v_new * dt
   ```

3. **隐式欧拉（Implicit Euler）**：最稳定，但计算成本较高
   使用定点迭代法近似求解未来状态

### 弹簧类型

| 弹簧类型 | 连接方式 | 刚度系数 | 作用 |
|---------|---------|---------|------|
| 结构弹簧（Structural） | 水平/垂直相邻 | k_s | 维持布料基本形状 |
| 剪切弹簧（Shear） | 对角线相邻 | 0.5*k_s | 防止剪切变形 |
| 弯曲弹簧（Bending） | 隔一点连接 | 0.1*k_s | 增强弯曲刚度 |

### 球体碰撞检测

当质点进入球体内部时，执行以下碰撞响应：

1. 将质点位置投影到球面上：`x = center + normal * radius`
2. 消除速度的法向分量：`v = v - (v · normal) * normal`

## 实现的功能

### 基础功能

- ✅ 质点弹簧模型初始化
- ✅ 结构弹簧（水平和垂直方向）
- ✅ 三种数值积分方法切换
- ✅ 布料固定点设置（左上角和右上角）

### 选做功能

- ✅ 剪切弹簧（对角线方向）
- ✅ 弯曲弹簧（隔点连接）
- ✅ 球体碰撞检测与响应

### 扩展功能

- ✅ 阻尼系数可调（1.0 / 5.0）
- ✅ 球体位置和半径可实时调整

## 使用方法

### 环境要求

- Python 3.8+
- Taichi 1.7+

### 运行命令

```bash
python 质点弹簧模型.py
```

### 操作说明

#### 控制面板

- **积分方法选择**：
  - `Explicit Euler (Explosive)`：显式欧拉，易发散
  - `Semi-Implicit Euler (Stable)`：半隐式欧拉，稳定（推荐）
  - `Implicit Euler (Damped)`：隐式欧拉，最稳定

- **阻尼系数切换**：
  - `k_d=1.0`：低阻尼，运动活跃
  - `k_d=5.0`：高阻尼，快速稳定

- **其他按钮**：
  - `Pause Simulation` / `Resume Simulation`：暂停/恢复模拟
  - `Reset Cloth`：重置布料到初始状态

#### 键盘控制

- `W/S`：球体上下移动
- `A/D`：球体左右移动
- `Q/E`：球体前后移动
- `R/F`：调整球体半径（R增大，F减小）

#### 鼠标控制

- 按住 `RMB`（右键）拖动：旋转视角
- 滚轮：缩放视角

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| N | 20 | 布料网格分辨率（N×N） |
| mass | 1.0 | 质点质量 |
| dt | 5e-4 | 时间步长 |
| k_s | 10000.0 | 弹簧劲度系数 |
| k_d | 1.0 / 5.0 | 阻尼系数 |
| gravity | (0, -9.8, 0) | 重力加速度 |
| max_velocity | 50.0 | 速度上限（防止数值爆炸） |
| sphere_radius | 0.3 | 球体半径 |

### 效果说明

运行程序后，可以观察到以下效果：

1. **布料自然下落**：布料从顶部固定点开始，在重力作用下自然下落
2. **弹簧约束效果**：结构弹簧维持布料形状，剪切弹簧防止菱形变形，弯曲弹簧使布料更加平整
3. **球体碰撞**：布料与球体接触时会产生碰撞响应，布料会包裹在球体周围
4. **阻尼对比**：
   - `k_d=1.0`：布料摆动明显，需要较长时间稳定
   - `k_d=5.0`：布料运动平缓，快速达到平衡状态
